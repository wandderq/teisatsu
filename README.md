# Teisatsu

**Teisatsu** - это утилита, которая позволяет разыскивать данные о любых входных данных, которые вы ей дадите.


## Примеры команд
 - Простой поиск: `teisatsu find google.com`
 - Поиск с отладкой: `teisatsu find  -v`
 - Просмотр списка доступных скриптов: `teisatsu list scripts`



## Принцип работы
1. Классификация: Входные данные проходят через **классификатор** - список функций, который присваивает входным данным **теги**.

2. Запуск скриптов и динамичное выведение результатов: по ранее найденным тегам отбираются нужные **скрипты**, которые ищут информацию о входных данных. В этоь же момент отображаются результаты выполнения скриптов.



## Установка
1. Через **pip** (проще всего)
    ```bash
    pip install git+https://github.com/wandderq/teisatsu@main
    ```

2. Через исходный код (для разработки)
    - Клонируем репозиторий
        ```bash
        git clone https://github.com/wandderq/teisatsu
        cd teisatsu
        ```
    
    - Создаем venv (рекомендовано)
        ```bash
        python -m venv .venv
        source .venv/scripts/activate
        ```
    
    - Устанавливаем пакет
        ```bash
        pip install .
        ```
